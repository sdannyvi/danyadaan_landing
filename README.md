A clean, minimal personal landing page for **Dan Yadaan** (דן ידען)—faculty, researcher, and consultant in AI and machine learning. Built with plain HTML and CSS, ready to host on **GitHub Pages**.

## Contents

- `index.html` — page structure and content
- `style.css` — layout, typography, and subtle animations
- This README

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Deploy with GitHub Pages

1. **Create a repository** on GitHub named `danyadaan-landing` (or any name you prefer).

2. **Push this project** to the repository:

   ```bash
   git init
   git add index.html style.css README.md
   git commit -m "Initial landing page"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/danyadaan-landing.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**: In the repo on GitHub, go to **Settings** → **Pages**.

4. Under **Build and deployment** → **Source**, select **Deploy from a branch**.

5. Choose branch **`main`** and folder **`/` (root)**, then save.

6. After a short build, your site will be available at  
   `https://YOUR_USERNAME.github.io/danyadaan-landing/`  
   (URL follows your username and repository name.)
